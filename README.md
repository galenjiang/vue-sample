### 目录介绍
* dist 项目输出路径
* src 项目编译路径

  * js js
    * core 公共js
    * lib 第三方框架
  * css css
    * core 公共css
  * sass sass文件
* gulpfile.js gulp配置文件
* package.json node配置文件
* webpack.config.js webpack配置文件
* README.md 说明文档
